VisionVoice AI
AI-powered visual understanding & natural voice narration for the visually impaired.

Overview

VisionVoice AI is an assistive technology tool that helps visually impaired users hear the world around them.
By combining Google Cloud Vision API with ElevenLabs TTS, the system converts images into clear, human-friendly voice descriptions.

Users can:

Upload an image

Capture a photo using a webcam

Instantly receive a spoken narration describing the scene

This project aims to enhance accessibility, independence, and everyday navigation for visually impaired individuals.

Features

✔ Object Detection — Identifies items like balls, phones, chairs, etc.
✔ Scene Understanding — Uses labels + web entities for rich descriptions
✔ Human-like Narration — Powered by ElevenLabs
✔ Webcam Snapshot Mode — Easy real-time picture capture
✔ Clean, Friendly Description Engine — Narration rewritten for clarity
✔ Audio Caching — Saves bandwidth + API credits
✔ Simple and Accessible UI
